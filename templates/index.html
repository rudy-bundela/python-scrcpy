Setup Instructions

1. Enable Developer Options: Go to Settings → About phone → Tap "Build number" 7 times

2. Enable USB Debugging: Go to Settings → Developer options → Enable "USB debugging"

3. Enable Wireless Debugging: In Developer options → Enable "Wireless debugging"

4. Get Pairing Info: Tap "Wireless debugging" → "Pair device with pairing code"

5. Use Connection Port: The main wireless debugging screen shows IP:Port for connection
